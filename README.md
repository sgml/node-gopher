node-gopher
==================

A Gopher server, written in node.

<hr>

Currently handles:

* Directories
* Text files
* Wav/MP3 Files
* Binary files
* GIF files

Future Additions:

* gophermap
* Support for search-indexing

<hr>

*WARNING*: THIS IS ENTIRELY A WORK IN PROGRESS - PLEASE, DO NOT USE THIS IN ANYTHING CRITICAL!

Not that you would...it's Gopher, after all...
